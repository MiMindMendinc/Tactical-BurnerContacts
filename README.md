# Tactical Burner Contacts

**Offline Burner Contact Manager** — Zero cloud. Zero tracking. Zero leaks.

Built for cops, fixers, journalists, activists, and anyone who can't afford their contacts living in the cloud.

### Features
- 100% client-side (localStorage only — nothing ever leaves your device)
- Contacts auto-delete after exactly 30 days
- Panic wipe: shake your phone 3 times within 1.5 seconds → everything gone
- One-tap vCard QR codes (library loads once, then works fully offline)
- Instant search across name, phone, notes
- JSON export
- Installable PWA (add to home screen like a real app)
- Clean tactical dark theme, fully responsive
- No frameworks, no telemetry, no backdoors

Companion tool to **[Tactical-FieldKit](https://github.com/MiMindMendinc/Tactical-FieldKit)**.

### How to Use
1. Download or open `index.html`
2. Open it on your phone (Chrome/Safari)
3. Tap **Share → Add to Home Screen** (it installs as a standalone app)
4. Start adding contacts — they never touch the internet
5. Grant motion permission once for the shake-to-wipe feature

Works in airplane mode forever.
